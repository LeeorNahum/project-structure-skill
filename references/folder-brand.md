# Brand Folder

Use `Brand/` when a project has versionable visual identity assets that should be pulled, reused, and updated over time.

Preferred shape for now:

```text
Brand/
└── Assets/
    ├── <Project> Icon/
    └── <Project> Logo/
```

Use this for source files and canonical exports for icons, logos, and closely related brand materials. Keep one-off screenshots, generated presentation exports, and temporary mockups outside `Brand/` unless they become part of the durable identity system.

Ask before renaming brand assets, flattening icon/logo folders, or mixing unrelated media into `Brand/Assets`.
